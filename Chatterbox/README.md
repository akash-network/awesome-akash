Chatterbox TTS
Chatterbox TTS is a lightweight, self-hosted Text-to-Speech system designed for easy deployment on Akash Network. 

This example provides a reliable backend API for converting text to speech, along with a simple frontend interface to interact with the basic functions.

What's Included

This repository contains everything you need to deploy Chatterbox TTS on Akash:

Backend: The core TTS API that handles text-to-speech requests

Frontend: A minimal web interface for testing and using the TTS functionality

Deployment
The main deploy.yaml file will create both the Chatterbox backend and API, as well as a simple frontend to interact with basic functions.

Individual Deployments

If you prefer to deploy components separately:

Backend Only: The backend directory includes the Chatterbox backend deploy.yaml for deploying the TTS API on its own.

Frontend Only: The frontend directory includes the frontend deploy.yaml for deploying the web interface on its own.

Requirements

Akash Network account

Access to Akash Console

Sufficient AKT tokens for deployment
