# Space Agent on Akash Network

## Overview

[Space Agent](https://github.com/agent0ai/space-agent) is an open-source AI agent built by the [Agent Zero](https://agent-zero.ai) team that runs in the browser layer and can reshape its own workspace on the fly. Unlike traditional AI assistants locked into a fixed UI, Space Agent can build pages, tools, widgets, and workflows directly into the running environment while you work.

## Key Features

- **Self-extending workspace** — the agent builds new capabilities into the UI in real time
- **Frontend-native runtime** — runs in the browser layer, directly interacting with the same framework it reshapes
- **Skill-based modularity** — new capabilities live in simple `SKILL.md` files the agent can write itself
- **Token-efficient** — no bulky tool-call JSON; actions stay in plain text and JavaScript
- **Personal to team-scale** — works as a solo assistant or scales to hierarchical multi-user/group deployments
- **Git-backed history** — roll back changes without taking down the whole system

## Deploy on Akash

This SDL deploys Space Agent on [Akash Network](https://akash.network), a decentralized cloud compute marketplace, giving you a self-hosted instance without relying on centralized cloud providers.

### Quick Start

1. Go to [Akash Console](https://console.akash.network) or to [Air Akash](https://air.akash.network/)
2. Fund your wallet with AKT and convert some to ACT
3. Deploy using the SDL in this directory
4. Access your Space Agent instance via the provided endpoint
